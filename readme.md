# Disaster Relief Management System (DRMS)

## 📌 Description
Disaster Relief Management System (DRMS) is a Flask-based web application designed to manage disaster response coordination efficiently.  
It helps manage relief centers, volunteers, victims, supplies, donations, and requests in one centralized platform.

---

## 🚀 Features

### 👥 User Roles
- Admin Dashboard
- Volunteer Dashboard
- Donor Dashboard
- Victim Dashboard

### 📂 CRUD Operations
- Manage relief centers
- Manage volunteers
- Manage victims
- Manage supplies
- Manage requests

### 📊 Reporting
- Low supplies report
- Top volunteers report

### 📝 Task Assignment
- Assign volunteer tasks
- Track progress

### 🌐 Web Interface
- HTML templates (templates/)
- Static assets (static/)

---

## 🛠 Tech Stack

- Backend: Flask (Python)
- Database: Configured in `db.py`
- Frontend: HTML, CSS, JavaScript
- Dependencies: See `requirements.txt`
